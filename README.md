SD2013-UIUC
===========

Peking Solar Decathlon 2013 - UIUC

## Setup

### (If) on Debian Server

- `sudo apt-get install python-pip python-dev libevent-dev`

### (If) on OSX Server

- `sudo apt-get install pip`

### Install pip and virtualenv

- `cd SD2013-UIUC/server`
- `virtualenv2 .`
- `bin/pip install -r requirements.txt`

### Running

- `cd SD2013-UIUC/server`
- `bin/python2 app.py`
